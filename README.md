# QGIS Complete Report Plugin

## Setup

Create virtual environment

- `python -m venv venv`
- `pip install pb_tool`

To activate the environment

- `source venv/bin/activate`


## Deploy

Run the command below to deploy to your local QGIS instance

- `pb_tool deploy -p ~/.local/share/QGIS/QGIS3/profiles/default/python/plugins`
